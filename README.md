Projek ini adalah sistem informasi surat untuk kecamatan Pantai Labu menggunakan:
- Frontend: React / Vite
- Backend: Node.js + Express + Prisma + SQLite/PostgreSQL
- API: CRUD surat, autentikasi, dan manajemen data.
